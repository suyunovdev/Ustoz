# EXAM — UMUMIY QOIDALAR

**Maqsad:** Loyihaga qo'shilish uchun amaliy ko'nikmalarni tekshirish  
**Vaqt:** 3 soat  
**Natija:** Tugallangan vazifa GitHub'da ko'rinishi kerak

---

## Qoidalar

1. Har bir talaba **faqat o'ziga berilgan vazifani** bajaradi
2. Google, dokumentatsiya ishlatish mumkin — ChatGPT/AI **taqiqlangan**
3. Vazifa **GitHub'da pull request** sifatida topshiriladi
4. Kod ishlashi shart — `npm run dev` xatosiz ishlasin
5. Vaqt tugaganda qilingan qismni ham topshirish mumkin

---

## Topshirish tartibi

```bash
git clone https://github.com/suyunovdev/Ustoz.git
cd Ustoz
git checkout -b exam/talaba-ism-familiya
# Vazifani bajaring
git add .
git commit -m "exam: [vazifa nomi] - Ism Familiya"
git push origin exam/talaba-ism-familiya
# GitHub da Pull Request oching
```
